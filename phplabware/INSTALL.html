Requirements

  • Apache Webserver (1.3.19 or higher).
  • php (4.04 or higher).
  • SQL server: PostgreSQL (7.0 or higher), or MySQL (3.23 or higher)

Linux distributions will include these. On MacOSX, download php and MySQL or Postgres from: Marc Ilianage's page. It also work on Windows (Yes, we tried..). Make sure your webserver, php, and SQL server are well-configured. Make sure you have a valid user account on your SQL server that can create databases.


Update Instructions

  • Make a backup of your database (for postgres: pg_dump phplabware > backupfile, for mysql mysqldump --opt phplabware > backupfile).
  • Did you make a backup of the database?
  • Unpack and decompress the phplabware folder: tar -zxvf phplabware_x.xxx.tar.gz (where x.xxx indicates the version number), 'over' your existing phplabware folder (replacing all the existing php files).
  • Login as sysadmin and run the script 'setup.php' (untill you do so the scripts and database are outof sync).

Installation Instructions

  • Unpack and decompress the phplabware folder: tar -zxvf phplabware_x.xxx.tar.gz (where x.xxx indicates the version number).
  • Place the phplabware folder in a web-accessible folder.
  • Edit the file includes/config_inc.exp and save it as includes/config_inc.php.
  • Create the database with the name you entered during the previous step
    (in postgres for instance: createdb -U postgresuser phplabware,
    in mysql; mysql -u mysqluser -p mysql, CREATE DATABASE phplabware;).
    Consult the documentation to your database server for details.
  • Point your browser to the script 'setup.php'. For security reasons this can only be done from the localhost.
  • Enter a password for the 'sysadmin' and submit the form.
  • login as user 'sysadmin' with the password you just entered.
  • Check the form with system settings. Most defaults should be fine. You can change these later too.
  • To convert uploaded MSWord documents into html you can use the library wvHtml.
  • For security reasons you might want to make sure that 'includes/config_inc.php' can only be read by your webdaemon, and that all files in directory phplabware are owned by root, readable by the webdaemon and not writeable by anyone but root. Make sure, however, that the webdaemon can write to directory files.
  • You can customize the welcome page by adding image files to the directory frontims.
