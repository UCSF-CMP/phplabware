Installation Instructions

  • Place the phplabware folder in a web-accessible folder on a well-configures server able to serve php scripts and with a SQL server.
  • Edit the includes/config_inc.exp and save it as 'includes/config_inc.php'.
  • Create the database with the name you entered during the previous step (in postgres for instance: createdb phplabware, in mysql; mysql mysql, CREATE DATABASE phplabware;). Consult the documentation to your database server for details.
  • Point your browser to the script 'setup.php'. For security reasons this can only be done from the localhost.
  • Enter a password for the 'sysadmin' and submit the form.
  • login as user 'sysadmin' with the password you just entered.
  • For security reasons you might want to make sure that 'includes/config_inc.php' can only be read by your webdaemon.
